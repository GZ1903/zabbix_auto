# Zabbix 一键部署

**脚本适用于操作系统：CentOS7.X/RedHat7.X部署，内存1G内存，磁盘容量大于5G,部署环境最好使用单独的服务器，避免环境冲突。**

#### 环境介绍：

```
Linux localhost.localdomain 3.10.0-1160.el7.x86_64 #1 SMP Mon Oct 19 16:18:59 UTC 2020 x86_64 x86_64 x86_64 GNU/Linux
5.5.68-MariaDB
Apache/2.4.6 (CentOS)
PHP 7.0.33
```

#### 执行脚本：

```shell
git clone
```
